# Zerch

A vector search engine for logs.

## Architecture(will update)

```
Zerch/
├── crates/
│   ├── zerch-core      
│   ├── zerch-embed     
│   ├── zerch-storage   
│   └── zerch-engine    
```

## Getting Started

Build the project:

```bash
cargo build
```

Run the engine:

```bash
cargo run -p zerch-engine
```
